# 🛍️ Myntra Clone

A fully responsive **Myntra Homepage Clone** built using only **HTML5** and **CSS3**. This project was created to strengthen my frontend fundamentals by recreating the layout and responsiveness of one of India's most popular e-commerce websites.

The project focuses on writing clean, semantic HTML, maintaining a scalable folder structure, and building responsive layouts using modern CSS techniques such as **Flexbox**, **CSS Grid**, and **Media Queries**—without using any JavaScript or CSS frameworks.

---

## 📖 About the Project

The primary goal of this project was to practice real-world frontend development by implementing a production-like user interface from scratch.

Instead of relying on libraries like Bootstrap or Tailwind CSS, every section was handcrafted using vanilla HTML and CSS to gain a deeper understanding of page layouts, alignment, spacing, responsiveness, and maintainable code organization.

This project demonstrates the ability to:

- Build responsive web pages from scratch.
- Structure HTML semantically.
- Organize CSS into reusable modules.
- Implement complex layouts using Flexbox and Grid.
- Follow clean coding practices.
- Maintain a professional project structure suitable for larger applications.
